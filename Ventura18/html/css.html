Preguntas y respuestas sobre CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que controla la presentación visual de un sitio web. Permite modificar colores, tamaños, márgenes, posiciones, fondos, tipografías y prácticamente cualquier aspecto del diseño de una página. Mientras HTML estructura el contenido, CSS transforma esa estructura en un diseño atractivo y profesional.

2. ¿Para qué sirve CSS?

CSS sirve para:

  • Dar estilo a los elementos HTML
  • Cambiar colores, tamaños, alineaciones y formas
  • Crear diseños adaptables a celulares (responsive)
  • Aplicar animaciones y transiciones
  • Separar el diseño de la estructura para un código más limpio
  • Crear interfaces modernas y profesionales

3. ¿Cómo funciona CSS?

CSS funciona mediante reglas que afectan elementos HTML. Estas reglas se aplican siguiendo el principio del “cascading” (cascada): si dos estilos afectan un mismo elemento, el navegador aplica el más específico o el último declarado. Cada regla tiene un selector, propiedades y valores.

4. ¿Dónde se coloca CSS?

  • CSS en línea: dentro de una etiqueta HTML usando “style”.
  • CSS interno: dentro de la etiqueta <style> en el <head>.
  • CSS externo: en un archivo .css independiente (forma profesional).

5. ¿Qué son los selectores en CSS?

Los selectores sirven para elegir qué elementos serán modificados. Hay selectores por etiqueta, clase, id, atributo, estado (hover), descendencia y más.

6. ¿Cuál es la diferencia entre clase e id?

class: se puede usar muchas veces. id: solo una vez por página. class = estilo, id = identificación.

Preguntas sobre HTML, CSS y su relación

1. ¿HTML puede funcionar sin CSS?

Sí, pero se vería simple y sin estilo. CSS hace que una página luzca moderna, limpia y atractiva.

2. ¿CSS puede modificar elementos HTML?

No modifica la estructura, pero sí cambia completamente la apariencia visual de esos elementos.

3. ¿Qué pasa si dos estilos afectan al mismo elemento?

El navegador aplica el estilo que tenga mayor especificidad o el que se encuentre más abajo en la cascada.

4. ¿Qué es responsive design?

Es la capacidad de una página para adaptarse a celulares, tablets y computadoras usando CSS (con flexbox, grid y media queries).

5. ¿Cuál es la diferencia entre HTML, CSS y JavaScript?

HTML = estructura CSS = diseño JavaScript = interacción Juntos forman cualquier sitio moderno.
