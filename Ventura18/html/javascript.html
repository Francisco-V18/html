Preguntas y respuestas sobre JavaScript

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interacción, dinamismo y lógica a las páginas web. Es un lenguaje que el navegador ejecuta para realizar acciones avanzadas, como mover elementos, validar formularios, mostrar mensajes, animar objetos o crear aplicaciones completas.

2. ¿Para qué sirve JavaScript en una página web?

JavaScript sirve para agregar comportamiento al sitio. Con él podemos:

  • Validar formularios
  • Crear animaciones y efectos
  • Modificar texto, imágenes y estilos al instante
  • Crear calculadoras, chats y juegos
  • Interactuar con bases de datos usando APIs
  • Manejar eventos como clics, teclas y movimientos

3. ¿Dónde se escribe JavaScript?

  • En línea: dentro de un elemento HTML
  • Interno: dentro de <script> en el mismo archivo HTML
  • Externo: en un archivo .js (la forma más recomendada)

4. ¿Cómo funciona JavaScript?

JavaScript se ejecuta gracias a un motor del navegador (como V8 o SpiderMonkey). Este motor lee el código línea por línea, interpreta las instrucciones y modifica el HTML o CSS según las órdenes del script. Con el DOM (Document Object Model) puede cambiar cualquier parte de la página.

5. ¿JavaScript puede modificar HTML en tiempo real?

Sí. JavaScript puede cambiar textos, imágenes, botones, añadir elementos nuevos o eliminarlos. Esto lo logra a través del DOM, accediendo a los elementos por id, clase o etiqueta.

Preguntas sobre HTML y su funcionamiento

1. ¿Qué es HTML y por qué es importante?

HTML es el lenguaje que crea la estructura de una página web. Organiza textos, imágenes, botones, formularios y todo el contenido visible. Sin HTML, los navegadores no podrían mostrar correctamente una página.

2. ¿Qué son las etiquetas HTML?

Las etiquetas son bloques de código que indican qué tipo de contenido se está mostrando: título, párrafo, tabla, imagen, encabezado, sección, etc.

3. ¿Cómo trabajan juntos HTML, CSS y JavaScript?

HTML estructura la información, CSS la diseña y JavaScript le da interacción. Los tres lenguajes son la base del desarrollo web moderno.

4. ¿Qué es el DOM?

El DOM (Document Object Model) es una representación del HTML que JavaScript utiliza para leer, modificar o eliminar cualquier elemento de la página.
